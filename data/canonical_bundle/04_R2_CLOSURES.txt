==SECTION:: 04.structural.unit.reversal ==

The r=2 unit-model exact symbolic block-structure / invariant-subspace
theorem.

Mechanism:
  The unit abstract matrix S(A,B,C,D,E) [ref: 03.abstract.r2.unit_S]
  commutes with the reversal involution sigma = (1 6)(2 5)(3 4):
    sigma * S = S * sigma

  Consequence:
    The +1 and -1 eigenspaces of sigma are 3-dimensional and are
    S-invariant. Restriction blocks:
      M_plus  (symmetric 3D restriction)
      M_minus (antisymmetric 3D restriction)
    with:
      det(y*I - M_plus)  = (y - L0) * Q(y)
      det(y*I - M_minus) = (y - L1) * Q(y)

  Therefore:
    det(y*I - S) = (y - L0) * (y - L1) * Q(y)^2

[STATUS: Verified Fact]
[GUARD: this mechanism does NOT transfer to the full r=2 model;
        see [ref: 06.failed_candidates] entry "full reversal candidate"]
[GUARD: this mechanism does NOT transfer to r=3 without independent
        witness]
[GUARD: Q(y)^2 is compact-factor multiplicity from two independent
        3D blocks; NOT numerical eigenvalue multiplicity]

==SECTION:: 04.structural.full.equitable ==

The r=2 full-model exact symbolic block-structure / invariant-subspace
theorem.

Mechanism:
  Equitable partition of the index set {1, ..., 9}:
    A = {1, 3, 5}
    B = {2, 4, 6}
    C = {7, 8, 9}

  This partition is equitable for the full abstract matrix
  T(X0,X1,X2,X3,X4) [ref: 03.abstract.r2.full_T] over Q[X0..X4].

  Cell-constant quotient block Q satisfies:
    det(y*I - Q) = (y - L0) * (y - L1)^2

  The zero-sum residual complement is invariant because T is symmetric
  and the cell-constant subspace is invariant.

  Residual block R satisfies:
    det(y*I - R) = C(y)^2

  Therefore:
    det(y*I - T) = (y - L0) * (y - L1)^2 * C(y)^2

Failed alternative partition:
  {1,2,3} | {4,5,6} | {7,8,9}
  failure witness: row 1 Cell A sum - row 2 Cell A sum = X2 - X4 != 0
  classification: Failed Candidate. Not equitable.

[STATUS: Verified Fact]
[GUARD: existence of the equitable partition is Verified Fact;
        uniqueness is recorded separately at
        [ref: 04.structural.partition.uniqueness]]
[GUARD: this mechanism does NOT transfer to r=3 without independent
        witness]
[GUARD: C(y)^2 is residual-block compact-factor multiplicity; NOT
        numerical eigenvalue multiplicity]

==SECTION:: 04.structural.unified.narrow ==

A narrow unified r=2 quotient-residual framework.

Both r=2 structural mechanisms admit a shared abstract description:

  - decompose the index set into invariant blocks (eigenspace blocks
    for unit; cell-constant quotient + zero-sum residual for full)
  - obtain block-diagonal characteristic polynomial structure
  - factor multiplicities arise from the dimensions of the invariant
    blocks rather than from genuine eigenvalue degeneracy

This shared architecture is the narrow unified framework. It is
canonical at this narrow level.

[STATUS: Verified Fact (narrow unified framework)]
[GUARD: this does NOT imply that the unit and full mechanisms are
        instances of a single, more rigorous abstract mechanism in
        the strong sense (block isomorphism, common group action,
        common representation). See
        [ref: 04.structural.distinctness.generic]]

==SECTION:: 04.structural.distinctness.generic ==

Generic structural distinctness of the unit and full quotient-residual
blocks.

Statement:
  The unit r=2 reversal-involution decomposition and the full r=2
  equitable-partition decomposition are GENERICALLY distinct
  mechanisms. There is no canonical map (embedding, projection,
  compression, or specialization in the generic sense) between the
  two mechanisms that turns one into the other while preserving the
  abstract-matrix structure.

Witness:
  (recorded in 16_STRUCTURAL_THEOREMS of the prior canonical;
   exact witness involves dimension-counting and label-ring argument
   showing no equivariant injection exists between the relevant block
   ambient spaces over Q[X0..X4])

[STATUS: Verified Fact]
[GUARD: this rules out a "stronger" unification in the generic
        equivariant sense; it does NOT rule out coincidence under
        artificial label specializations]

==SECTION:: 04.structural.partition.uniqueness ==

  is UNIQUE as an equitable partition of T(X0,X1,X2,X3,X4) viewed
  over the ring Q[X0,X1,X2,X3,X4].
