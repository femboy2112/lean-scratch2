classification :

scope :
  Reconnaissance over the closed unit/full r=3 S-level characteristic

  Ubar_s(y) | Fbar_s(y) is rejected at all three slices.

  R_unit_s != R_full_s at all three slices. Own row-sum factor checks
  pass: U_s(R_unit_s) = 0, F_s(R_full_s) = 0. Cross row-sum checks

target :

classification :
  Computational Observation only.

scope :
  Strategy Candidate C (B-level normalization scoping) per
  [ref: 09.strategy.proposed_v27_candidates].

scaling_rule :
  B = c_3 * S, where c_3 = h^2 / (1 - t^54)^2.
  For p_S(y) = y^n + a_1 y^{n-1} + ... + a_n,
  p_B(x) = c_3^n * p_S(x / c_3) = x^n + (a_1 c_3) x^{n-1} + ... + (a_n c_3^n).
  Coefficient k from the leading term: coeff_B[k] = coeff_S[k] * c_3^k,
  with k=0 for the leading coefficient.

mode_full :
  Full sidecar mode emits all 19 (unit) or 28 (full) B-level
  coefficients with hash anchors.

mode_k1 :
  Minimal-certificate mode tests only the trace coefficient
  (coefficient at degree-1 from leading, k=1). Sufficient to block
  descent-to-Q at the first non-leading coefficient if nonrational.
